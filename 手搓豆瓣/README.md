# 手搓豆瓣 - 豆瓣电影网站复刻项目

## 项目简介

这是一个仿照豆瓣电影桌面版网站开发的个人项目，使用原生HTML、CSS和JavaScript实现，展示了电影信息、搜索、筛选等核心功能。

## 功能特点

### 🎬 核心功能
- **正在热映**：横向滚动展示当前热映电影，支持购票跳转
- **热门电影**：网格布局展示，支持分类筛选（热门、最新、高分、冷门）
- **热门电视剧**：多分类筛选（综合、国产、综艺、欧美、日韩、动画、纪录片）
- **一周口碑榜**：排行榜展示，TOP10电影推荐
- **热门片单**：精选电影合集推荐
- **热门影评**：优质影评内容展示

### 🔍 交互功能
- **智能搜索**：支持电影名称、导演、演员、类型搜索
- **分类筛选**：Tab切换，实时数据更新
- **响应式设计**：适配桌面、平板、手机等多种设备
- **图片懒加载**：优化页面加载性能
- **平滑滚动**：提升用户体验

### 🎨 界面设计
- **豆瓣风格**：还原豆瓣电影的经典界面设计
- **现代化UI**：卡片式布局，阴影效果，悬停动画
- **配色方案**：绿色主色调，符合豆瓣品牌色
- **字体优化**：使用系统字体，确保中文显示效果

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **布局**：CSS Grid + Flexbox
- **响应式**：媒体查询，移动端优先
- **数据**：JSON模拟数据，支持异步加载
- **工具函数**：防抖、节流、懒加载等优化

## 快速开始

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd 手搓豆瓣
   ```

2. **直接运行**
   由于使用纯前端技术，可以直接在浏览器中打开 `index.html` 文件。

3. **本地服务器（推荐）**
   使用本地服务器可以获得更好的开发体验：
   ```bash
   # 使用Python
   python -m http.server 8000

   # 或使用Node.js
   npx http-server -p 8000
   ```

4. **访问项目**
   打开浏览器访问 `http://localhost:8000`

## 功能演示

### 搜索功能
- 在顶部搜索框输入电影名称、导演、演员等关键词
- 支持实时搜索，输入即显示结果
- 搜索结果会显示在热门电影区域

### 分类筛选
- 热门电影区域支持分类切换：热门、最新、高分、冷门
- 电视剧区域支持多维度筛选
- 点击Tab按钮即可切换分类

### 响应式体验
- 桌面端：多栏布局，充分利用大屏幕空间
- 平板端：自适应布局，保持良好的阅读体验
- 手机端：单列布局，触摸友好的交互设计
