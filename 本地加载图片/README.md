# 图片批量加载工具

一个用于批量加载和查看图片URL的工具，特别适用于风险排查时查看客户门头图或签收图。

## 功能特点

- ✅ **双模式输入**：支持手动输入URL和Excel文件批量导入
- ✅ **批量加载**：一次性加载所有图片，无需逐个打开网页
- ✅ **状态监控**：实时显示图片加载状态（成功/失败/加载中）
- ✅ **数据持久化**：自动保存URL列表到本地存储
- ✅ **响应式设计**：适配不同屏幕尺寸
- ✅ **图片预览**：点击图片可在新标签页查看大图

## 使用方法

### 方式一：手动输入URL
1. 在"URL链接"标签页中输入图片URL
2. 点击"添加URL"或按回车键
3. 重复添加所有需要查看的URL
4. 点击"加载所有图片"

### 方式二：Excel文件导入
1. 切换到"Excel文件"标签页
2. 点击"选择Excel文件"
3. 选择包含图片URL的Excel文件（支持.xlsx和.xls格式）
4. 工具会自动提取所有有效的URL链接
5. 点击"加载所有图片"

## Excel文件格式要求

- 支持任意列包含URL链接
- 工具会自动扫描所有单元格中的URL
- 支持多个工作表，默认使用第一个工作表

## 启动方式

直接在浏览器中打开 `index.html` 文件即可使用。

## 技术栈

- HTML5 + CSS3 + JavaScript
- SheetJS (xlsx.js) - Excel文件解析
- 本地存储 - 数据持久化

## 注意事项

1. 请确保图片URL可公开访问
2. 部分图片可能因跨域限制无法加载
3. 建议每次加载的图片数量不要过多（建议不超过50张）
4. 工具会自动保存URL列表，下次打开时可直接使用

## 使用场景

- 风险排查时批量查看客户门头图
- 批量查看签收图片
- 其他需要批量查看图片URL的场景
