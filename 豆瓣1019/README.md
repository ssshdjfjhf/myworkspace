# 个人电影网站

一个类似豆瓣电影的个人电影展示网站

## 功能特性

- ✅ 电影列表展示
- ✅ 正在热映/即将上映
- ✅ 电影详情页
- ✅ 搜索和筛选
- ✅ 评分展示
- ✅ 响应式设计

## 技术栈

- 前端：HTML5, CSS3, JavaScript
- 后端：Node.js + Express
- 数据：JSON文件存储

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

```bash
npm start
```

### 3. 访问网站

打开浏览器访问：http://localhost:3000

## 项目结构

```
.

├── data/              # 数据文件
```
