# 个人电影网站 - 使用指南

## 🎉 恭喜！你的电影网站已经搭建完成！

---

## 📋 项目概述

这是一个类似豆瓣电影的个人电影展示网站，具有以下功能：

### ✨ 核心功能
- ✅ **电影展示**：首页展示正在热映、即将上映、高分电影
- ✅ **电影详情**：点击电影卡片查看详细信息
- ✅ **搜索功能**：支持按电影名、导演、演员搜索
- ✅ **筛选功能**：按类型、地区筛选电影
- ✅ **响应式设计**：支持手机、平板、电脑访问
- ✅ **美观界面**：豆瓣绿色主题，现代化设计

## 🎯 功能使用说明

### 1. 浏览电影

- **首页**：自动加载正在热映、即将上映、高分电影
- **滚动浏览**：向下滚动查看更多内容
- **点击卡片**：点击任意电影卡片进入详情页

### 2. 搜索电影

- 在顶部搜索框输入关键词
- 支持搜索：电影名、导演名、演员名
- 按回车或点击搜索按钮

### 3. 筛选电影

在"豆瓣高分"区域：
- **按类型筛选**：剧情、喜剧、动作、科幻、动画等
- **按地区筛选**：美国、中国、日本、韩国等
- 点击筛选按钮即可过滤结果

### 4. 查看详情

点击电影卡片后可以看到：
- 电影海报
- 评分和评价人数
- 导演、主演信息
- 类型、地区、年份
- 片长
- 剧情简介

## 🔧 自定义和扩展

### 添加更多电影

编辑 `data/movies.json` 文件，按照现有格式添加新电影：

```json
{
  "id": 21,
  "title": "电影名称",
  "originalTitle": "Original Title",
  "rating": 9.0,
  "ratingCount": 1000000,
  "poster": "海报图片URL",
  "year": 2024,
  "directors": ["导演名"],
  "actors": ["演员1", "演员2"],
  "genres": ["类型1", "类型2"],
  "duration": 120,
  "regions": ["地区"],
  "summary": "剧情简介...",
  "status": "released",
  "tags": ["标签1", "标签2"]
}
```

### 修改样式

编辑 `public/css/style.css` 文件：

- **主题色**：搜索 `#00b51d` 替换为你喜欢的颜色
- **字体**：修改 `font-family` 属性
- **布局**：调整 `.movie-grid` 的 `grid-template-columns`

### 修改端口

编辑 `server.js` 文件，修改第8行：

```javascript
const PORT = 3000;  // 改为其他端口，如 8080
```

## 📊 API接口说明

后端提供以下API接口：

### 获取所有电影
```
GET /api/movies
参数：
  - genre: 类型筛选
  - region: 地区筛选
  - sort: 排序方式 (rating/year/hot)
  - search: 搜索关键词
```

### 获取电影详情
```
GET /api/movies/:id
```

### 获取正在热映
```
GET /api/movies/status/now-playing
```

### 获取即将上映
```
GET /api/movies/status/coming-soon
```

### 获取高分电影
```
GET /api/movies/status/top-rated
```

### 获取所有类型
```
GET /api/genres
```

### 获取所有地区
```
GET /api/regions
```

## 🔍 常见问题

### Q: 如何添加更多电影？
A: 编辑 `data/movies.json` 文件，按照现有格式添加新的电影对象。

### Q: 电影海报不显示怎么办？
A: 检查海报URL是否有效，或使用本地图片路径。

### Q: 如何修改网站标题？
A: 编辑 `public/index.html` 和 `public/detail.html` 的 `<title>` 标签。

### Q: 可以部署到服务器吗？
A: 可以！将项目上传到服务器，安装Node.js，运行 `npm install` 和 `npm start` 即可。

### Q: 如何修改端口？
A: 编辑 `server.js` 文件中的 `PORT` 变量。
