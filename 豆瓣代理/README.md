# 豆瓣电影代理服务器

⚠️ **重要声明**: 此项目仅用于技术学习和演示目的。未经授权代理他人网站可能违反相关法律法规和服务条款。

## 功能特性

- ✅ 代理豆瓣电影网站内容
- ✅ 自动修复资源链接(图片、CSS、JS)
- ✅ 在页面中注入自定义按钮和功能
- ✅ 添加自定义下载入口
- ✅ 浮动工具栏

## 安装步骤

1. 安装依赖:
```bash
npm install
```

2. 启动服务器:
```bash
npm start
```

3. 访问代理网站:
```
http://localhost:3000
```

## 开发模式

使用 nodemon 自动重启:
```bash
npm run dev
```

## 自定义功能

### 1. 修改注入的按钮样式
编辑 `server.js` 中的 CSS 样式部分

### 2. 添加下载链接
在 `/download` 路由中实现实际的下载逻辑

### 3. 注入更多功能
在 cheerio 处理部分添加更多 HTML/JS 注入

## 技术栈

- Node.js
- Express
- Axios (HTTP 请求)
- Cheerio (HTML 解析和修改)
